# OBS Atom ⚛️

OBS Atom - is a lightweight and highly optimized streaming studio built for older or low-spec Linux systems. 

Stream to Twitch, YouTube, Instagram, Facebook and more at smooth 720p quality, even on a 2GB Aspire One. 

**Features:**  
- Auto-tune settings based on system hardware  
- First-run configuration for optimal performance  
- Tray menu integration for quick access (GTK AppIndicator / C++ Qt)  
- Supports Twitch, YouTube, TikTok Live, and more  
- Optimized for low-spec machines while retaining professional-quality streaming  

**Installation:**  
Clone the repository and follow the `INSTALL.md` for full setup instructions, or run the `.deb` installer included in releases.
